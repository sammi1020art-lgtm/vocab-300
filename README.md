# 英文 300 單字學習遊戲

互動式英文單字學習網頁，含字卡、連連看、拼字、聽力、口說等模式，支援學生進度、打卡、記事本與老師後台管理。

## 線上使用

開啟 `index.html` 或部署至 GitHub Pages 後，使用瀏覽器直接開啟即可。

## 帳號（預設）

| 角色 | 帳號 | 密碼 |
|------|------|------|
| 老師 | Ab100000 | Ab100000 |
| 學生 Jim | Jim | 1021029 |
| 學生 Sherry | Sherry | 1031217 |

學生以**英文姓名**登入（不分大小寫）。

## 功能

- 301 個內建單字，11 個學習主題
- 五大練習模式：單字卡、連連看、拼字、聽力、口說
- 老師後台：學生進度、帳號管理、Excel 上傳自訂單字主題
- 雲端同步（Tinkr）：跨裝置同步學習進度

## 老師上傳自訂單字

1. 老師登入後台
2. 上傳 Excel（第一欄英文、第二欄中文）
3. 檔名即為學習主題名稱

## 本地開發

本專案為單一 HTML 檔案，無需建置步驟。直接以瀏覽器開啟 `300.html` 或 `index.html`。

## GitHub Pages 部署

1. 將此 repo 推送到 GitHub
2. 前往 **Settings → Pages**
3. Source 選 **Deploy from a branch**，Branch 選 `main`，資料夾選 `/ (root)`
4. 儲存後即可透過 `https://<username>.github.io/<repo>/` 存取
